<h2>🔨 Tutorial de execução da aplicação:</h2>

<h3>Antes de mais nada: Instale as dependências</h3>
<p>É necessário ter Python e GIT instalados no seu sistema para que a aplicação possa rodar.</p>
<p>Para baixar o Python, acesse <a href="https://www.python.org/downloads">https://www.python.org/downloads</a>.</p>
<p>Para baixar o Git, acesse <a href="https://git-scm.com/downloads">https://git-scm.com/downloads</a>.</p>

<br>

<strong>1)</strong> Faça o download deste repositório com o comando git clone
<pre><code>git clone https://github.com/maarantes/portfolio_digital_dsm.git</code></pre>

<strong>2)</strong> Abra o terminal na pasta API e crie um Ambiente Virtual
<pre><code>python -m venv venv</code></pre>

<strong>3)</strong> Ative o Ambiente Virtual
<pre><code>.\venv\Scripts\activate</code></pre>

<strong>4)</strong> Saia da pasta API para ir na pasta mãe onde o requirements.txt se encontra
<pre><code>cd ..</code></pre>

<strong>5)</strong> Instale o requirements.txt
<pre><code>pip install -r requirements.txt</code></pre>

<strong>5)</strong> Vá para a pasta API novamente
<pre><code>cd api</code></pre>

<strong>6)</strong> Rode a Aplicação
<pre><code>python app.py</code></pre>

<strong>7)</strong> Acesse o link abaixo para visualizar o site
<pre><code>http://127.0.0.1:5000</code></pre>
